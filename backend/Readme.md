### VectorShift Backed

#### Tech Stack

- Python - FastAPI
- uvicorn
- Docker

#### Application Scripts

_Note: Python version used :- 3.11.0_

- Create and activate a virtual environment

```
python3 -m venv .venv
source .venv/bin/activate
```

- Install dependencies

```
pip install -r requirements.txt
```

- Start the Backend server

```
uvicorn main:app --reload
```
